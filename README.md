# PacketPrism

PacketPrism is a software-based enterprise network observability and security controller.

## Objective
- Monitor live network traffic
- Detect abnormal traffic behavior
- Enforce security policies automatically

## Architecture
- Rust Agent: Traffic capture and analysis
- Dashboard: Centralized monitoring & control
- Automation: Policy enforcement logic
